### DATA AVAILABILITY STATEMENT

The data and metadata analyzed and generated in this research project and referenced by the source code are publicly available at [https://doi.org/10.5281/zenodo.20135209](https://doi.org/10.5281/zenodo.20135209). The associated terms of use for data and metadata are specified in a separate document also available at the link provided above.
The files in the data bundle release have been constructed in order to regenerate the original workspace and preserve the meaning of relative file paths reported in the metadata registries. Source files (i.e. PDFs of scientific publications) have been uploaded in a different release at [https://doi.org/10.5281/zenodo.20137704](https://doi.org/10.5281/zenodo.20137704).

The source code available in this repository references the expected data layout and can be used to reproduce the data output available at the link provided above.
